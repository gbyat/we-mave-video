## [1.0.0] - 2026-06-10

### Added

- Self-hosted `@maveio/components` player bundle with themes and locale dependencies
- Shortcode `[we_mave_player embed="…"]` with global defaults and per-embed overrides
- Optional Gutenberg block and Enfold-friendly snippet generator
- Admin settings for component update schedule, CDN debug mode, and player defaults
- GitHub release workflow, changelog tooling, and optional plugin self-updates
- Translation tooling (POT/JSON generation via WP-CLI)
